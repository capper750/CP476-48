# Finance Tracker

A simple web-based personal finance tracker that allows users to add transactions, view transaction history, and analyze spending by category.

This project is built using HTML, CSS, and JavaScript with browser localStorage used for temporary data persistence.

## Features

### Dashboard
- Displays total income
- Displays total expenses
- Displays net balance
- Shows most recent transactions

### Add Transaction
- Add income or expense
- Select category
- Add date and description
- Form validation
- Saves to localStorage

### History
- View all transactions
- Filter by date range
- Delete transactions

### Category Summary
- Shows total spending by category
- Expenses only
- Sorted highest to lowest
