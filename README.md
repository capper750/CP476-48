# Finance Tracker

A simple web-based personal finance tracker that allows users to add transactions, view transaction history, and analyze spending by category.

This project is built using HTML, CSS, and JavaScript with browser localStorage used for temporary data persistence.

1. ensure node.js and npm are installed on your laptop
2. Open terminal and run the repository from the terminal
